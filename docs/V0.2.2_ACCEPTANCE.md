# v0.2.2 acceptance evidence

Date: 2026-07-11 EDT

## Scope

v0.2.2 is a dependency and CI maintenance release. It preserves the v0.2.1
PAN-OS tools, authorization, inventory, mutation state, permitted XPath roots,
and deployment behavior. Multi-vsys and HA work remains paused pending suitable
lab resources, and Panorama remains deferred.

## TLS, authentication, and MCP reads

certificate for `rust-panosmcp.mechub.org`, valid from 2026-07-11 through
2026-10-09. Anonymous and invalid-token requests return HTTP 401. The configured
reader credential authenticates, and an MCP 2025-03-26 session initialized
against server version 0.2.2.

Read-only MCP acceptance then passed both required calls:

- `list_devices` returned target `panosvm`, endpoint `192.168.1.225`, and
  `vsys1`;
- `gather_device_facts` returned model `PA-VM`, serial `007054000270810`, and
  PAN-OS `12.1.5`.

The separately configured writer and reviewer credentials intentionally
expired before this dependency-only rollout and correctly return HTTP 401.
They were not rotated as part of v0.2.2; privileged credential issuance remains
a separate operator action.
